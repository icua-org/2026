# 2026 International Conference on Urban Analytics

프로페셔널한 국제 학술 컨퍼런스 웹사이트

## 🎯 프로젝트 개요

ICUA 2026 (International Conference on Urban Analytics)은 도시 분석, 빅데이터, 스마트 시티 기술 분야의 선도적인 연구자들과 실무자들이 모이는 국제 학술대회입니다. 본 웹사이트는 학회의 전문성과 학술적 권위를 효과적으로 전달하는 모던하고 반응형 디자인으로 제작되었습니다.

## 📅 학회 정보

- **일시**: 2026년 6월 19-20일 (2일간)
- **장소**: KAIST (Korea Advanced Institute of Science and Technology), 대전, 대한민국
- **주제**: Advancing Urban Science through Data, Technology, and Innovation

## 🚀 사용 방법

### Google Blogger에 업로드

1. **HTML 복사**
   - `index.html` 파일의 내용을 복사합니다

2. **Blogger 설정**
   - Blogger 대시보드 > 테마 > 편집 HTML
   - 전체 코드를 붙여넣기

3. **CSS/JS 호스팅**
   - CSS와 JS 파일을 별도 호스팅 (예: GitHub Pages, Cloudflare Pages)
   - HTML의 링크 경로를 업데이트

### 또는 독립 호스팅

1. 모든 파일을 웹 호스팅 서버에 업로드
2. index.html을 루트에 배치
3. 브라우저에서 접속

## 🔧 커스터마이징

### 색상 변경
CSS 파일의 `:root` 변수를 수정:
```css
:root {
    --primary-color: #1e3a8a;
    --secondary-color: #3b82f6;
    --accent-color: #f59e0b;
}
```

### 연사 정보 추가/수정
HTML 파일에서 `.speaker-card` 섹션을 편집

### 프로그램 일정 수정
`.timeline-item` 블록을 추가/수정
